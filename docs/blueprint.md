# **App Name**: Been Oy Cleaning Services

## Core Features:

- Homepage Presentation: Showcase Been Oy's mission, values, and core competencies through a compelling homepage narrative.
- Service Catalog: Provide a detailed list of cleaning services offered, including descriptions and possible customization options.
- Contact Form: Implement a contact form for potential clients to send inquiries and request quotes.
- AI-Powered Quote Tool: Use generative AI to analyze customer requirements (as expressed in the contact form or otherwise) and automatically formulate a preliminary service quote, including pricing and a range of options. This is presented to the staff as a suggested plan that they can review and forward.

## Style Guidelines:

- Primary color: Light blue (#ADD8E6) to evoke cleanliness and trust.
- Background color: Off-white (#F5F5F5) for a clean and uncluttered feel.
- Accent color: A slightly darker shade of blue (#87CEEB) for interactive elements and call-to-action buttons.
- Body and headline font: 'Inter' sans-serif for a modern, clean, and readable look.
- Use simple, outline-style icons in blue to represent different cleaning services and company features.
- Employ a grid-based layout for organized and responsive design.
- Subtle transition animations on navigation and form interactions to enhance user experience.
